# Projet Statistical Learning — Prédiction de la courbe des taux US

M1 Mathématiques et Applications (Statistiques), Université Paris Dauphine.
Projet 1 — régression supervisée sur données FRED.

## Reproductibilité

1. Cloner le repo.
2. Créer un environnement virtuel : python -m venv .venv puis .venv\Scripts\activate
3. Installer les dépendances : pip install -r requirements.txt
4. Ouvrir les notebooks : jupyter lab
5. Exécuter les notebooks dans l'ordre 01_ à 06_.
